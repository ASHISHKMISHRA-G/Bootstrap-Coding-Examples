Absolute Position - It removes element from normal flow of document. - It keeps the element with regard to the content in page. - It moves along with the content. [top, left, right, bottom] Fixed Position - It removes element from normal flow. - It keeps the element with regard to the browser. - It is fixed at a specific position in browser. - It locks scrolling.
